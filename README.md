# Movie Recommendation System

A Movie Recommendation System built using Python and data analysis techniques that suggests movies based on user ratings and preferences. This project was developed during an Online Internship at Acmegrade.

# Project Features

Data preprocessing and cleaning

Exploratory Data Analysis (EDA)

User-Movie rating matrix creation

Movie similarity using collaborative filtering

Personalized movie recommendations

# Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

# Dataset

The project uses:

Movies dataset

User ratings dataset

# How to Run the Project

Clone the repository

Install required libraries

pip install pandas numpy matplotlib seaborn

Run the notebook
Open Movie_Recommendation_System.ipynb and run all cells.
